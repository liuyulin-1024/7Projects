# 表单引擎原型复现文档

## 📋 项目概述

本项目是对 Axure 原型（http://www.axshare.site/QQKAWM）的完整复现，包含了表单引擎系统的全部17个页面。

- 修订历史记录

### 2. 字段模版库
- 字段管理列表
- 字段同步功能
- 使用次数统计

### 3-4. 数据源管理
- 数据源列表管理
- 新增/编辑数据源配置
- 数据库连接测试

### 5-6. 建模管理
- 数据模型列表
- 新增/编辑模型
- 主表和明细表配置

### 7. 表单管理
- 表单列表
- 版本管理
- 发布状态管理

### 8. 表单编辑器（核心页面）
- 左侧组件面板
- 中间画布区域
- 右侧属性配置面板
- 拖拽式表单设计

### 9. 需增加/调整组件
- 组件开发需求说明
- 优先级划分

### 10-13. 组件配置（弹窗页面）
- 公式配置

### 14. 表单样式管理

- 系统内置函数
- 自定义函数管理

### 16-17. 组件库
- 组件列表管理
- 新增/编辑自定义组件

## 🚀 使用方法

### 本地预览

1. 打开 `index.html` 文件即可在浏览器中查看
2. 无需安装任何依赖
3. 支持所有现代浏览器

### 导航方式

- **左侧导航栏**：点击页面名称切换页面
- **键盘快捷键**：使用 ↑ 和 ↓ 键在页面间切换
- **URL Hash**：支持 `#page-01` 到 `#page-17` 直接访问

## 📐 设计规范

### 颜色方案
- 主色调：`#1890ff`（蓝色）
- 成功色：`#52c41a`（绿色）

- 次要文本：`#8c8c8c`（灰色）
- 边框色：`#d9d9d9`（浅灰）
- 背景色：`#fafafa`（浅色背景）

### 按钮规范
- **主要按钮**（btn-primary）：蓝色背景，白色文字
- **次要按钮**（btn-default）：白色背景，灰色边框

### 表格规范
- 表头：浅灰色背景
- 行高：48px
- 边框：1px 实线，浅灰色
- hover 效果：浅蓝色背景

## 📝 核心页面说明

### 表单编辑器（Page 8）
这是系统最核心的页面，采用三栏布局：

**左侧组件面板（280px）**
- 组件库分类
- 拖拽式添加组件
- 基础组件展示

**中间画布区（自适应）**
- 表单预览区域
- 实时编辑
- 选中组件高亮

**右侧属性面板（320px）**
- 字段属性配置
- 公式配置入口
- 数据联动设置
- 隐藏/必填条件

### 公式配置系统（Pages 10-13）
统一的公式编辑界面，支持：
- 双面板布局（字段列表 + 函数库）
- 拖拽或双击插入
- 实时语法验证
- 多种运算符支持

## 🛠️ 技术栈

- **HTML5**：语义化标签
- **CSS3**：Flexbox 布局、Grid 布局
- **JavaScript**：原生 ES6+
- **字体**：系统默认字体栈

## ⚡ 性能优化

1. **单页面架构**：所有页面在一个 HTML 文件中，快速切换
2. **CSS 复用**：统一的样式类，减少代码重复
3. **轻量级**：无第三方依赖，加载速度快

## 🔄 后续开发建议

### 前端框架迁移
建议使用以下技术栈进行实际开发：
- **Vue.js 3.x + Ant Design Vue** 或
- **React 18+ + Ant Design**

### 关键功能模块
1. **表单设计器**：参考 form-generator、form-create
2. **公式引擎**：实现类似 Excel 的公式解析
3. **拖拽系统**：使用 Sortable.js 或 Vue.Draggable
4. **代码编辑器**：集成 Monaco Editor 或 CodeMirror

### 数据流设计
- 状态管理：Vuex / Pinia 或 Redux
- 数据持久化：LocalStorage + IndexedDB
- 实时协作：WebSocket
