## 🔍 注释标记系统说明

### 注释标记的特征

从截图和页面快照数据中，我们可以看到：

1. **红色圆圈数字标记**（如 ①、②、③）
   - 位置：在输入框、按钮、下拉框等组件旁边
   - 作用：标识需要说明的功能点
   
2. **黄色问号标记**（如 ?）
   - 位置：在标题或字段名旁边
   - 作用：表示需要特别说明的概念

3. **原型说明区域**
   - 位置：页面底部，通常有黄色背景
   - 内容结构：
     - 原型说明：总标题
     - 标签说明 + 编号
     - 功能说明 + 编号
     - 操作说明 + 编号
     - 页面说明（包含页面名称、页面描述、功能应用）

## 🛠️ 提取工作流程

### 步骤1：查看截图，识别注释标记
```
1. 打开对应页面的截图
2. 找出所有红色圆圈数字和黄色问号
3. 记录标记的编号和位置
```

### 步骤2：从页面底部提取说明文字
```
1. 定位到黄色背景的"原型说明"区域
2. 按照结构提取：
   - 标签说明 X.X 内容
   - 功能说明 X.X 内容
   - 操作说明 X.X 内容
   - 页面说明内容
```

### 步骤3：更新HTML文件
```html
<!-- 在对应页面的 HTML 中添加标记 -->
<input type="text" class="note-marker" data-note="1">
<button class="note-marker" data-note="2">按钮</button>

<!-- 在页面底部添加说明区域 -->
<div class="prototype-notes">
    <h3>原型说明：</h3>
    <div class="note-section">
        <h4>标签说明 <span class="note-num">1</span></h4>
        <p>具体说明内容...</p>
    </div>
</div>
```

### 步骤4：验证和调整
```
1. 在浏览器中打开 index.html
2. 切换到对应页面
3. 对照截图检查是否有遗漏
4. 调整样式使其与原型一致
```

## 📦 内容提取工具建议

### 方法1：手动提取（推荐用于小规模）
- 优点：准确度高
- 缺点：耗时较长
- 适用：2-3个复杂页面

### 方法2：OCR文字识别
```python
# 可以使用 Python + Tesseract 批量提取截图中的文字
from PIL import Image
import pytesseract

# 提取注释区域的文字
image = Image.open('screenshot.png')
text = pytesseract.image_to_string(image, lang='chi_sim')
```

### 方法3：使用浏览器DevTools
```
1. 在原始 Axure 原型中打开浏览器开发者工具
2. 复制 iframe 内的 HTML 结构
3. 提取其中的文字内容和注释说明
```
