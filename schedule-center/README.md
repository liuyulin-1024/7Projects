# 日程中心静态页面

这是一个日程中心管理系统的静态页面原型复现。

## 页面列表

### PC端页面
1. **index.html** - 日程中心主页
   - 包含待办中心卡片
   - 日历视图
   - 日程列表
   - 需求说明面板

2. **calendar.html** - 日历详细页面
   - 完整日历视图
   - 日程管理

3. **subscribe.html** - 订阅广场
   - 订阅管理

### 移动端页面
1. **mobile/index.html** - 移动端首页
2. **mobile/day-view.html** - 日视图
3. **mobile/month-view.html** - 月视图

## 交互说明

### 日历交互
1. **日期选择**：点击日期数字，该日期置实心蓝底
2. **月份切换**：点击左右箭头切换月份
3. **创建日程**：
   - 点击右上角"+"按钮
   - 或右键点击任意日期
4. **日程标记**：
   - 红色圆点：未开始的日程
   - 灰色圆点：过期的日程

### 标签切换
- 点击"待办"、"已办"、"已发"、"办结"标签切换视图

### 悬停效果
- 待办事项卡片：向上浮动
- 日程项：向右移动并放大

## 样式规范

### 颜色方案
- 主色调：紫色渐变 (#667eea - #764ba2)
- 强调色：蓝色 (#1890ff)
- 警告色：红色 (#ff4d4f)
- 成功色：绿色 (#52c41a)
- 背景色：浅灰 (#f5f7fa)

### 圆角
- 卡片：12px
- 按钮：8px
- 小元素：4px

### 阴影
- 卡片：0 2px 8px rgba(0,0,0,0.08)
- 悬停：0 4px 12px rgba(102, 126, 234, 0.15)

## 使用方法

1. 直接在浏览器中打开 `index.html`
2. 无需服务器，所有功能均为静态实现
3. 建议使用现代浏览器（Chrome、Firefox、Edge等）

## 注意事项

1. 图片资源需要放置在 `images/` 目录下
2. 日程数据目前为静态示例数据
3. 实际使用时需要连接后端API获取真实数据
4. 移动端页面需要在移动设备或浏览器开发者工具的移动视图中查看
